# RestAPI

| HTTP Verb | Path(route)  | Do               |
| --------- | ------------ | ---------------- |
| GET       | /student     | 得到所有學生資料 |
| POST      | /student/:ID | 新增學生資料     |
| PUT       | /student/:ID | 覆蓋學生資料     |
| PATCH     | /student/:ID | 更新部分學生資料 |
| DELETE    | /student/:ID | 刪除學生資料 |
* 網頁瀏覽器只能送出GET和POST，其他要用postman測試
